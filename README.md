# 匠单 CraftOrder 参赛包

项目定位：非遗传承人的研学课程与体验接单工作台。

推荐赛道：社会服务  
推荐附加标签：社会公益 / 非遗创新创意开发

## 文件说明

- `index.html`：可直接打开的静态 Demo。
- `styles.css`：页面视觉和响应式样式。
- `app.js`：交互逻辑和示例整理规则。
- `submission-kit.md`：报名帖、初赛帖、短视频脚本和提交检查表。
- `evidence-notes.md`：政策与真实场景依据，用于作品帖和答辩。
- `trae-session-record.md`：TRAE Session ID 记录表和关键 Prompt。

## 初赛 Demo 体验路径

1. 打开 `index.html`。
2. 先看顶部“接单流程”，了解课程包、报价单、预约页和传播包的关系。
3. 在“工作台”选择非遗项目、城市、客群、时长、人数和材料成本。
4. 点击“整理方案”，查看方案记录、课程与定价建议、可交付成果包，并点击“复制成果包”。
5. 展开“定价测算详情”，确认材料、场地、人工、设备折旧和城市系数如何影响建议价。
6. 依次查看“研学课”“预约页”“传播包”“依据库”。
7. 在“预约页”填写日期、时段和联系方式，提交后查看预约确认状态。
